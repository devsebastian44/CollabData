# **Nombre de la App**: CollabData

## Características Principales:

- **Autenticación de Usuarios**: Cuentas de usuario seguras con funcionalidad de inicio de sesión/registro.
- **Carga de Conjuntos de Datos (Datasets)**: Los usuarios pueden cargar datasets en formatos CSV, XLSX o JSON.
- **Proyectos Compartidos**: Los usuarios pueden crear y gestionar proyectos compartidos para el análisis colaborativo.
- **Selección Dinámica de EDA**: Los usuarios pueden seleccionar funciones de EDA (estadísticas descriptivas, histogramas, diagramas de caja, diagramas de dispersión, mapas de calor) desde la interfaz de usuario para que se realicen en sus datasets.
- **Insights de Datos Impulsados por IA**: La aplicación proporcionará sugerencias para análisis basados en el contenido del dataset, los tipos de datos de las columnas y los patrones generales, utilizando una herramienta que determina si cada pieza de información debe incorporarse o no.
- **Visualización de Resultados**: Muestra los resultados del análisis utilizando Matplotlib y Seaborn dentro de la aplicación.
- **Arquitectura Escalable**: Una arquitectura de backend modular diseñada para escalar de manera eficiente usando FastAPI y Pandas.

## Directrices de Estilo:

- **Color Primario**: Azul (#135bec), reflejando la naturaleza colaborativa y centrada en la ingeniería de la aplicación.
- **Color de Fondo**: Gris claro (#f6f6f8) para un aspecto limpio y moderno; fondo del modo oscuro: Gris oscuro (#101622).
- **Color de Acento**: Turquesa (#2dd4bf) como color de acento contrastante y análogo para elementos interactivos.
- **Fuente del Cuerpo**: 'PT Sans' (sans-serif) para un cuerpo de texto cálido y legible.
- **Fuente de los Títulos**: 'Playfair' (serif) para un título elegante.
- **Material Symbols** para un diseño moderno y consistente.
- **Diseño basado en rejilla responsiva** para una visualización óptima en diferentes dispositivos.
- **Transiciones sutiles** para elementos interactivos, como botones y elementos de menú.
